# 🎨 React Native Expo UI con NativeWind

Este proyecto es una **interfaz de usuario (UI)** desarrollada con **React Native** y **Expo**, que implementa un sistema de navegación moderno utilizando **Drawer** y **Tabs**, estilizado con **NativeWind** (Tailwind CSS para React Native). Es una base sólida para desarrollar el frontend de aplicaciones móviles con navegación estructurada y estilos personalizados.

---

## 🚀 Características

- 🔁 Navegación con **Drawer** y **Tabs** usando `expo-router`
- 🎨 Estilos rápidos y responsivos con **NativeWind**
- ⚛️ Interfaz construida 100% en **React Native**, lista para conectar con cualquier backend
- 📦 Estructura organizada, ideal para escalar

---

## 📦 Requisitos

- Node.js ≥ 16
- Expo CLI (`npm install -g expo-cli`)
- Yarn o npm
